# The Movie

---

### Skills

- React, Typescript, styled-components, firebase, Redux, motion

### Feature / Description

- the-movie apis를 이용하여 최신 혹은 인기 있는 `영화`, `TV프로그램`를 소개하는 애플리케이션

### Specific Feature

- `Login` : google, github를 통해 로그인
- `Navigation` : movie, tv, actors 구분
- `Home` : 인기있는 영화, TV프로그램 리스트
- `Detail` : 해당 프로그램 타이틀, 릴리즈 연도, 줄거리, 트레일로 등 상세페이지
- `검색기능` : TV,movie,actor 등 검색하는 기능
- `다크모드` 기능
